🟦 Day2 振り返りまとめ（API・通信・階層構造）

今日は、ReactとLaravelがどうやって会話しているかを、階層構造として理解した。

通信は上から下へ次の順で成り立っている。
まず HTTP / HTTPS という通信の道があり、その上で
リクエスト（React → Laravel）とレスポンス（Laravel → React） の往復が行われる。

リクエストには

HTTPメソッドで「システムにどうしてほしいか（取得・追加・更新・削除）」を伝え、

JSONで「何のデータを、どんな内容で扱うか」という中身を渡す。

Laravelは処理結果を JSON として返し、ReactはそのJSONを元に画面を作る。

👉 APIは、この階層構造の上で成り立つ「お願い窓口」だと理解できた。
